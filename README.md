# Zmodyfikowany wariant gry kółko i krzyżyk 4x4

Projekt z przedmiotu Sztuczna inteligencja.

## Opis

Gra kółko i krzyżyk z planszą 4x4, wygraną po 3 znakach w linii oraz agentem komputerowym opartym o algorytm Minimax z alfa-beta pruning i funkcją heurystyczną.

## Uruchomienie

```bash
py TicTacToe.py
```

## Pliki

- `TicTacToe.py` — kod gry
- `Dokumentacja-TicTacToe.docx` — dokumentacja projektu
